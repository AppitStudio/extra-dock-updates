VERSION: 4.0.0
DETAILS:
Our new major version update.
In this version we complete our widget system with new widgets and some core widgets,
So the updated widgets list are:
- Clock - A fully customizable clock that supports themes, sizes, timezone settings, and much more.
- IP - A widget that displays your public IP address right in your dock.
- Stripe - A widget that allows you to add a stripe restricted key to ExtraDock, that is safely stored in the keychain.
Then it can show you the your gross and net sales volumes for selected period of time.
All configurable from the widget settings.
And the core widgets are:
- Spacer - A widget that adds custom spacing between dock items to create the perfect layout that matches your workflow and aesthetic preferences.
- Finder - A widget that allows you to open the Finder application right from your dock.
- Divider - A widget that adds a divider between dock items to create a more professional look.
- Trash - A widget that allows you to have a trash right in your extra dock.

Also in this version we added an integration with DockFlow!
https://dockflow.appitstudio.com/
This integration allows you to user DockFlow presets switching to manage your visible docks
per preset. so you can set extra docks that will be visible for specific dockflow preset.

We have added a new onboarding experience and improved the overall UI and performance of the app.

More to come!
